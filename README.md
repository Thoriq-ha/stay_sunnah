## Stay, Stay Sunnah
 # Aplikasi Reminder Ibadah Sunnah

[Getx, Sqflite, Firebase, Push Notification, Dark Mode]
### Installation

_Below is a tutorial of how to use this app.

1. Clone the repo
   ```sh
   git clone https://github.com/Thoriq-ha/stay_sunnah.git
   ```
2. Run pub get
   ```sh
   flutter pub get
   ```
3. Run this project
   ```sh
   flutter run
   ```
